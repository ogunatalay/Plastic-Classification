The dataset collected by the research team consists of **5,600 images** of **7 different types of plastic**:  

- 🧴 **HDPE**  
- ♻️ **LDPE**  
- 🧪 **PVC**  
- 🍶 **PET**  
- 📦 **PP**  
- 🍽️ **PS**  
- 🗑️ **Other**
